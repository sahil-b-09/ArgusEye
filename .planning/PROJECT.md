# ArgusEye Trading Indicator Platform

## Core Value
ArgusEye provides premium trading indicators. The platform serves as their complete digital storefront and automated fulfillment system, allowing users to discover indicators, purchase access securely, and automatically receive access in their TradingView accounts without manual intervention.

## Current Focus
Establishing the v1 platform: a beautiful, high-converting landing page, seamless payment processing, and backend automation to grant TradingView script access.

## Technical Constraints & Preferences
- **Frontend**: Needs a premium, beautiful UI/UX (to be executed using the UIUX Pro Max skill).
- **Backend/Hosting**: Cloud hosting setup provided by the client. Needs database to capture user records.
- **Payments**: Indian payment gateway integration (e.g., Razorpay) to handle transactions.
- **Fulfillment**: Automated integration with TradingView to grant access to private invite-only scripts using the captured TradingView username.
- **User Data to Capture**: Full Name, Email ID, Phone Number, TradingView Username. Guest checkout flow heavily preferred for friction-less conversion.
